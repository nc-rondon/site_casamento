5b. Parallax Scroll Clip — .about-section

Background image with background-attachment: fixed. As the user scrolls, the hero section slides upward revealing the fixed background beneath — creating a cinematic clip/reveal transition between sections.

.about-section {
  background-image: url("...");
  background-attachment: fixed; /* ← KEY */
  background-size: cover;
  background-position: 50%;
}

5c. Full-bleed Image Reveal + Typography Overlay — .model-01-section

A 100vh section with overflow: hidden. The cinematic effect is created by three stacked layers: (1) the background photo, (2) a large SVG typography image ('Worn by / the Bold') absolutely positioned and spanning the full width, and (3) a font-texture image overlay (object-fit: cover, z-index: 2) that creates the depth effect where the model appears to be "inside" the text.

/* Layer stack (bottom → top) */

/* 1. Background photo */
.model-01-back-image { object-fit: cover; width: 100%; height: 100%; }

/* 2. Giant SVG typography — positioned absolute, spans full width */
.model-01-title-block { position: absolute; z-index: 2; width: 100%; justify-content: space-between; }
.model-01-title-image._01, ._02 { width: 36%; } /* ← SVG files */

/* 3. Font texture image — covers the photo, "reveals" through text edges */
.model-01-font-image {
  position: absolute; z-index: 2;
  object-fit: cover; min-width: 105%; min-height: 102%;
  pointer-events: none;
}
